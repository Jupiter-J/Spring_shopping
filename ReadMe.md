1. 회원 기능

   2. 회원 조회
2. 상품 기능
   1. 상품 등록
   2. 상품 수정
   3. 상품 조회
3. 주문 기능
   1. 상품 주문
   2. 주문 내역 조회

   2. 상품의 종류는 도서, 음반, 영화가 있다
   3. 상품을 카테고리로 구분할 수 있다
